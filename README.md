# sudoku
This is a program that solves Sudoku puzzles.

### Files

* _main.cpp_
* _gameh.h_
* _strategies.h_

Input puzzles must have 9 rows. Numbers are separated in each row by comma. Absent values are indicated by '0' or 'NA'.
